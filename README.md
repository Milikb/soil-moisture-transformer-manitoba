# soil-moisture-transformer-manitoba
Code, model settings, and processed outputs for Transformer-based soil moisture prediction across Manitoba, Canada.

This repository contains scripts, model settings, processed outputs, and figure data associated with the manuscript:

Transformer-Based Deep Learning Models for Soil Moisture Prediction across Manitoba, Canada

Raw soil moisture, meteorological, and groundwater monitoring datasets are not redistributed due to data-provider restrictions. These data should be requested directly from Manitoba Agriculture and Manitoba Environment and Climate Change. Publicly available CaSR data are available through https://doi.org/10.5065/N8PJ-1X12.

The repository provides the code and processed outputs used to reproduce the main model evaluation, mapping, and figures reported in the manuscript.
